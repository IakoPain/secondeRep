INTERET

C'est un langage universel qui permet de consulter des pages web quel que soit le support :

- ordinateur

- téléphone portable

- tablette

çA S'ECRIT COMMENT ?

Le langage HTML est du texte qui permet de structurer la page web .

Il s'écrit avec des balises qui fonctionnent toujours par deux, une balise d'ouverture et une de fermeture

Exemples :

<html> et </html>   pour indiquer que la page est écrite en HTML
<head> et </head> En-tête : informations générales sur la page qui ne s'afficheront pas sur le navigateur
<body> et </body> Corps : contenu de la page ( textes, images, liens ...)
