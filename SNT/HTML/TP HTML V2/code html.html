Hyper Text Markup Language ou Langage de Balisage d'Hypertext

C'est quoi l'Hypertext ?

Le concept d'Hypertexte numerique est inventee par le sociologue americain Ted Nelson en 1965.

C'est un texte numerique contenant des liens vers d'autres textes numeriques ainsi interconnectees.

Aujourd'hui, les liens hypertexte permettent de naviguer de page web en page web.

La source du lien peut-etre du texte comme des images, des videos ou des sons.
