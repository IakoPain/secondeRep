LE LANGAGE CSS

Le langage CSS ( Cascading Style Sheets ou feuilles de style en cascade) est ajouté au code HTML de la page pour mettre en forme la page web ( arrière-plan, type et taille des polices de caractères, bordures...)
Une modification d'un élément de la feuille de style CSS se répercute automatiquement à toutes les pages liées à cette feuille.

çA S'ECRIT COMMENT ?

Le langage CSS est un fichier texte que le navigateur ira consulter pour mettre en forme la page ouverte.

Il s'écrit avec des propriétés qu'il faut préciser.

Exemples :

background-color :white     L'arrière plan sera en blanc
font-family: "calibri",serif     La police de caractère sera calibri
font-size : 25px                    La taille sera de 25 px
